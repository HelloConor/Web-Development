Collapsible Set:

This is the first website I have worked on, I like to read so I thought this would be a good starting point. I started off with very simple HTML & CSS work. I have learned this mostly by thinking of ideas for the page, and the googling tutorials which explain how to go about doing that. I found that beginner friendly StackOverflow questions & w3schools.com to explain this very well.

Over time I started to introduce beginner Javascript projects into this website. I found Javascript to be much more abstract than HTML & CSS so I knew I had to keep it simple and with great documentation so I can read through it to remind me. I found Youtube & StackOverflow to be great resources on how to learn Javascript and how I can implement it into this website.
